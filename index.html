Entrega Secches

Tuve la idea de crear una página web para cargar/buscar discos.
Cuando el usuario hace click en "Cargar Nuevo Disco", se le pide que ingrese el nombre del disco, el nombre del artista, un código único que le permitirá buscarlo luego, y las pistas dentro del disco.
Luego, cuando el usuario hace click en "Mostrar Discos", se le muestran todos los discos que ingresó con toda la información.
Por último, cuando el usuario hace click en "Buscar Disco", se le pide que ingrese el código único del disco que quiere buscar, y se le muestra toda la información de ese disco en particular.
